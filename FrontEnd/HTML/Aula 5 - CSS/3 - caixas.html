ADICIONANDO CSS NO HTML

  • InLine - Atributo 'style na linha da TAG'
  • <style> - TAG no HEAD do HTML
  • <link> - link a um documento CSS externo
  • @import - regra do CSS, portanto, deve ser usada dentro do css, ao invés de dentro do HTML. NÃO recomendado pois deixa o site menos responsiva
